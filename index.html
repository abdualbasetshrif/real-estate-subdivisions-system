البرومبت الاحترافي 📋
قم بتحليل كروكي الأرض المرفق واستخرج المساحة الإجمالية للمخطط، والمساحة المقتطعة للطرق، بالإضافة إلى بيانات القطع العقارية بتنسيق JSON نظيف تماماً بدون أي شروحات أو نصوص مقدمة أو مؤخرة. التنسيق المطلوب: { "grossArea": 35000, "roadArea": 5000, "plots": [ { "id": 43, "area": 250.5, "facade": "شرقية", "nEastName": "44", "nWestName": "42", "nNorthName": "طريق 16م", "nSouthName": "38" } ] } شروط استخراج وتحليل البيانات الفنية الذكية: 1. تحديد الاتجاهات الجغرافية (البوصلة): - ابحث أولاً عن رمز البوصلة (وردة الرياح/مؤشر الاتجاهات الجغرافية) في كروكي أو صورة المخطط لتحديد اتجاه الشمال الحقيقي والشرق بدقة. - إذا وجد في الصورة أي نقطة (Point) أو خط (Line) تم وضعه للإشارة لاتجاه الشرق أو معلم جيوغرافيك، اعتمده فوراً كدليل لتحديد باقي الاتجاهات الأربعة. - في حال عدم وجود بوصلة أو مؤشرات اتجاهات واضحة في الصورة، اعتمد الاتجاه الافتراضي لصور الخرائط والمخططات: * أعلى الصورة = الشمال الجغرافي (North) * أسفل الصورة = الجنوب الجغرافي (South) * يمين الصورة = الشرق الجغرافي (East) * يسار الصورة = الغرب الجغرافي (West) 2. استخراج جيران القطعة الأربعة (Neighbors Map): - لكل قطعة أرض يتم استخراجها، حدد بدقة متناهية حدودها الجغرافية الأربعة (الشرق، الغرب، الشمال، الجنوب) وفقاً لاتجاهات البوصلة المحددة أعلاه: * nEastName: الجار الشرقي الملاصق مباشرة (ضع رقم القطعة المجاورة مثل "44" أو اسم الشارع مثل "طريق 16م"). * nWestName: الجار الغربي الملاصق مباشرة (ضع رقم القطعة المجاورة مثل "42" أو اسم الشارع). * nNorthName: الجار الشمالي/البحري الملاصق مباشرة (ضع رقم القطعة المجاورة مثل "5" أو اسم الشارع). * nSouthName: الجار الجنوبي/القبلي الملاصق مباشرة (ضع رقم القطعة المجاورة مثل "38" أو اسم الشارع). 3. استخراج البيانات الأساسية: - grossArea: إجمالي مساحة المخطط بالكامل بالمتر المربع (رقم فقط). - roadArea: مساحة الطرق المقتطعة بالمتر المربع (رقم فقط). - id: رقم القطعة (رقم صحيح فقط). - area: المساحة الصافية للقطعة (رقم عشري أو صحيح فقط). - facade: اتجاه الواجهة باللغة العربية بناءً على البوصلة والشارع الذي تطل عليه القطعة (مثال: "شرقية"، "شمالية"، "جنوبية غربية"). ملاحظة هامة: إذا لم تجد المساحة الكلية (grossArea) أو مساحة الطريق (roadArea) مكتوبة بوضوح في الكروكي، ضع قيمتهما 0 في الـ JSON، واستخرج القطع وجيرانها الأربعة بدقة كاملة.
