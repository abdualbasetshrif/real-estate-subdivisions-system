# 🏢 منظومة إدارة المقسمات والمخططات العقارية (Baseet Real Estate System)

تطبيق متكامل لإدارة المقسمات والمخططات العقارية، تتبع قطع الأراضي، المبيعات والحجوزات، المنظومة المحاسبية، ورأس مال المساهمين.

## ⚡ الميزات الرئيسية
- **داتابيز سحابية (Supabase Cloud)**: مزامنة فورية وحفظ أمن لكافة المقسمات والقطع والعقود والأقساط.
- **إدارة عقود البيع والحجوزات**: تسجيل عقود البيع وحساب المبالغ المدفوعة والمتبقية وتوليد خطط الأقساط.
- **لوحة تتبع الأقساط**: متابعة مواعيد استحقاق الدفعات وتسديد الأقساط بضغطة زر.
- **تحليل المخططات بالذكاء الاصطناعي**: وحدة قراءةخرائط ورسومات المخططات الهندسية أوتوماتيكياً عبر Gemini API.
- **خيارات دخول سريعة وفورية**: أزرار دخول بنقرة واحدة لمدراء الشركات والوكلاء والوسطاء بدون عوائق.

## 🚀 كيفية التشغيل محلياً
1. تشغيل الخادم عبر Node.js:
   ```bash
   node server.js
   ```
2. افتح المتصفح على:
   `http://localhost:8080`
